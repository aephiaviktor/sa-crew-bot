# SA Crew Bot

Desktop bot for Star Atlas Crew trading on Tensor.

## Setup

```bash
npm install
npm run build
npm start
```

Settings are stored by the Electron app in the user data directory. Sensitive values such as API keys, RPC URLs, and wallet secrets should be entered in the app settings, not committed to the repo.

## Development

- Source: `src/`
- Electron UI: `electron/`
- Build output: `dist/`
